### 3. Staking & Rewards

- **Stake ANM** to unlock higher tiers
- **Earn rewards** automatically (APY varies by tier)
- **Priority scheduling** based on staking amount
- **Unstake anytime** (rewards claimed first)

### 4. Governance & Evolution

The platform evolves **by the creators, for the creators**:

- **Create Proposals**: Minimum 100 ANM staked
- **Vote**: Voting power = staked ANM
- **Quorum**: 10% of staked tokens must vote
- **Approval**: 66% approval required
- **Voting Period**: 7 days

**What You Can Vote On:**
- New AI agent types
- Feature upgrades and improvements
- New style packs and cinematic expansions
- Platform parameters and economics
- Treasury allocations
- Ecosystem partnerships

### 5. Deflationary Economics

Every production consumes ANM with a 0.5% fee:
- **60% Burned**: Reduces supply, increases scarcity
- **40% Reinvested**: Funds platform development

**This creates constant deflationary pressure, increasing token value over time.**

## üöÄ Getting Started

### Installation

```bash
# Clone the repository
git clone https://github.com/YourOrg/AnimAIverse.git
cd AnimAIverse

# Install dependencies
pip install -r requirements.txt

# Run the ANIMA application
python anima_app.py
```

### Quick Start Example

```python
from anima_app import AnimAIverseANIMA

# Initialize ANIMA system
app = AnimAIverseANIMA()

# Onboard a new user with initial tokens
app.onboard_user("user_alice", initial_tokens=1000.0)

# Stake tokens to unlock features
app.stake_tokens("user_alice", 500.0)

# Create an animation
result = app.create_animation(
    user_id="user_alice",
    genre="sci-fi",
    theme="Space exploration",
    characters=["Captain", "Engineer", "Alien"],
    duration=5,
    resolution="1080p",
    fps=30,
    style="cinematic"
)

# Check your dashboard
dashboard = app.get_dashboard("user_alice")
print(f"Balance: {dashboard['wallet']['balance']} ANM")
print(f"Tier: {dashboard['tier']['current']}")
```

## üîß API Reference

### Token Operations

```python
# Get balance
balance = app.anima.get_balance("user_id")

# Transfer tokens
app.anima.transfer_tokens("from_user", "to_user", amount)

# Get token info
token_info = app.anima.get_token_info()
```

### Staking Operations

```python
# Stake tokens
app.stake_tokens("user_id", amount)

# Unstake tokens
app.unstake_tokens("user_id", amount)

# Claim rewards
app.claim_rewards("user_id")

# Get staking info
staking_info = app.anima.get_staking_info("user_id")
```

### Governance Operations

```python
from token.governance import ProposalType

# Create proposal
app.create_proposal(
    "user_id",
    title="Add Manga Style Pack",
    description="High-quality manga/comic style...",
    proposal_type="style_pack",
    proposal_data={"pack_name": "manga_pack", "styles": [...]}
)

# Vote on proposal
app.vote_on_proposal("user_id", "PROP-0001", vote_for=True)

# Get active proposals
proposals = app.get_active_proposals()
```

### Production Operations
